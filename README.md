# TechNova Store

Projeto simples de site desenvolvido para demonstrar a integração entre HTML, CSS, JavaScript e React. A proposta é uma loja fictícia com visual moderno, seção institucional e catálogo de produtos com busca e filtro em tempo real.

## Estrutura do projeto

- `index.html`: estrutura base da página.
- `styles.css`: estilos visuais, responsividade e organização do layout.
- `app.js`: lógica em React com busca, filtro por categoria e renderização da lista de produtos.

## Funcionalidade interativa

A principal funcionalidade do projeto é o catálogo com filtro em tempo real. O usuário pode digitar na barra de busca e também selecionar categorias. O conteúdo da lista é atualizado imediatamente com base no estado controlado pelo React.

## Como executar

1. Abra o arquivo `index.html` no navegador.
2. Se preferir, utilize uma extensão como Live Server no VS Code.
3. A conexão com a internet é necessária para carregar React, ReactDOM, Babel e as fontes externas.
